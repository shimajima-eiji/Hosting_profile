# 転職経験が多い理由
契約社員/派遣契約をしつつ、個人事業主としてプロダクトに参加しており、経験プロジェクト数は10件以上あります。
退職理由はプロダクトローンチであったり、内製メンバーへの移管完了がほとんどです。
